Tecnologías

WebGL / GLSL

10 fragment shaders procedurales: respiración, vértigo, fractales Mandelbrot, ruido Perlin 3D, ondas de choque.

Three.js r128

Capa 3D con 5 geometrías, 5 cámaras, 4 sistemas de iluminación, deformación de vértices, post-procesado.

SVG Animado

10 entidades vectoriales deformables con filtros de distorsión, parallax de 3 capas, movimientoBrowniano.

Canvas 2D

CRT sintético: estática, scanlines, barras de glitch, viñeta. + 80 partículas flotantes con física.

Web Audio API

Análisis RMS en tiempo real, 8 loops WAV procedural, crossfade primario/secundario reactivo.

Python Generator

Orquestador que genera HTMLs standalone únicos. ADN de movimiento, presets de director, 25 frases de horror.

Contenido Único Cada Vez

Cada ejecución del pipeline genera un video visual completamente único. ADN de movimiento aleatorio, combinaciones de shader + paleta + entidad + Three.js = infinitas variaciones.
